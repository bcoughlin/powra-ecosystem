# Beginner Guide

Welcome to Game Alpha! This guide will help you get started and understand the core mechanics.

## Understanding the Basics

### Core Gameplay Loop

1. **Explore** - Navigate the game world
2. **Battle** - Engage in combat
3. **Collect** - Gather resources and items
4. **Upgrade** - Improve your character

### Key Concepts

#### Stats

- **Health (HP)** - Your survivability
- **Attack** - Physical damage output
- **Defense** - Damage reduction
- **Speed** - Movement and attack speed
- **Magic** - Magical ability power

#### Resources

- **Gold** - Primary currency
- **Gems** - Premium currency
- **Energy** - Used for special abilities

## Your First Hour

### Tutorial Completion (10 minutes)

Follow the in-game tutorial to learn basic controls and mechanics.

### Early Game Strategy (50 minutes)

1. **Complete starter quests** - Easy way to gain levels and resources
2. **Unlock basic abilities** - Focus on your character's core skills
3. **Gather materials** - Collect common resources for crafting
4. **Join a guild** (optional) - Get help from experienced players

## Common Beginner Mistakes

### ❌ Don't Do This

- Spending premium currency on common items
- Ignoring the tutorial
- Trying to skip ahead too quickly
- Neglecting defensive stats

### ✅ Do This Instead

- Save premium currency for rare items
- Complete all tutorial missions
- Progress at a steady pace
- Balance offense and defense

## Tips & Tricks

::: tip Pro Tip
Always check the daily missions - they're an easy source of free resources!
:::

::: warning Important
Don't sell rare materials early in the game - you'll need them for crafting later!
:::
